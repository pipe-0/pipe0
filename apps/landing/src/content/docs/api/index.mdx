All endpoints require a bearer token. See <AppLink linkType="authentication">Authentication</AppLink>.

## OpenAPI specification

The full schema is published at [https://api.pipe0.com/openapi](https://api.pipe0.com/openapi). Use it to generate a typed client, or use the prebuilt [TypeScript client](/docs/sdks/typescript-client).
