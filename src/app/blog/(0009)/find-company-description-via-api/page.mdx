# Retrieve Company Info via API

In this blog post, we explore how you can generate a company description via an API endpoint.
For this, we use [pipe0](https://pipe0.com).

Pipe0 is a framework for data enrichment. To enrich data with pipe0 you use enrichment pipes.
There are several pipes available to generate a company description via API. For this blog post, we're 
using the pipe [CompanyGetDescriptionPipe0:Website](https://www.pipe0.com/resources/pipe-catalog/company-get-description-pipe0-website/version-1).

This pipe relies on scraping website data from a URL and summarizing this data with the help of an LLM.

The cost per invocation is ~1ct. per company description.

```typescript
const id = 1;
const result = await fetch("https://pipe0.com/api/v1/run", {
  method: "POST",
  headers: {
    "Authorization": `Bearer TOkEN`,
  },
  body: JSON.stringify({
    pipes: [{ name: "CompanyGetDescriptionPipe0:Website@1" }],
    input: [{id, companyWebsiteURL: "<COMAPNY_URL>"}]
  })
});

const response = await result.json();

// Print description
console.log(response.records[id].fields["companyDescription"].value);
```
